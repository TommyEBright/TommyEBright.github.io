Section 3

Normal values at rest.

These are the textbook ranges you need to know. Values change with age, fitness, and exercise intensity.

Measurement Normal range What it represents Clinical note
Heart rate 60–100 bpm Beats per minute at rest <60 = bradycardia; >100 = tachycardia
PR interval 120–200 ms Atrial depolarisation + AV node delay >200ms = first-degree heart block
QRS duration <120 ms Ventricular depolarisation >120ms = bundle branch block
QT interval 350–450 ms Ventricular depolarisation + repolarisation Corrected for HR (QTc); prolonged QT = arrhythmia risk
P wave <120 ms, <0.25 mV Atrial depolarisation Absent P wave = AF or junctional rhythm
R–R interval 600–1000 ms Time between successive beats Basis for HRV (heart rate variability) analysis
