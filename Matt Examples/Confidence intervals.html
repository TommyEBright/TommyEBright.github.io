Misconception

What is a Confidence Interval?

📊 Definition: A confidence interval (CI) is a range of plausible values for an unknown population parameter (e.g., mean, proportion) based on sample data. It provides a measure of precision and uncertainty.
🎯 Point Estimate
The single best guess from sample (e.g., sample mean).
🔄 Margin of Error
Half-width of CI = critical value × standard error.
✅ Correct interpretation: "We are 95% confident that the true population mean lies between LL and UL" — this reflects the method's long-run accuracy.

Formula & Components

CI = x̄ ± (critical value) × (s / √n)
📌 Critical value (z/t)
Depends on confidence level (1.96 for 95%).
📉 Standard Error
s/√n — smaller SE → narrower CI.
📊 Sample size (n)
Larger n = smaller SE = more precise.

Confidence Intervals vs. p-values

📊 Confidence Intervals
• Show magnitude and direction
• Indicate precision
• More informative than binary sig.
⚡ p-values
• Only indicate statistical significance
• Don't show effect size
• Can be misleading with large n
💡 Modern statistical thinking: Report CIs instead of or alongside p-values. A 95% CI that excludes the null value (e.g., 0) indicates p < 0.05, but also tells you how large the effect might be.

Test Yourself

Q1: A 95% CI for mean 100m sprint time is [10.5, 11.3] seconds. What does "95% confident" mean?
✅ If we repeated sampling many times, ~95% of such intervals would contain the true population mean.
Q2: If you increase sample size from 30 to 120, what happens to the CI width?
✅ Width decreases (narrower interval) because standard error = SD/√n gets smaller.
Q3: A 99% CI is narrower than a 95% CI. True or false?
❌ False — 99% CI is wider because you need more certainty (larger critical value).
